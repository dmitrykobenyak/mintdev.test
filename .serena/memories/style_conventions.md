# Style and Conventions

## Code Style
- **JavaScript**: ES6+ features, classes, arrow functions
- **CSS**: Modern CSS with custom properties, flexbox, grid
- **HTML**: Semantic HTML5 elements
- **Naming**: camelCase for JavaScript, kebab-case for CSS classes
- **File Structure**: Single-page application with separate HTML, CSS, JS files

## Design Guidelines
- **Colors**: Soft palette with teal, lavender, white
- **Typography**: Large, readable text
- **Layout**: Clean, minimal design
- **Animations**: Smooth transitions and micro-interactions
- **Responsive**: Mobile-first approach

## JavaScript Conventions
- Use classes for main application logic
- Event-driven architecture
- LocalStorage for data persistence
- Modern DOM manipulation methods
- Error handling for user interactions

## CSS Conventions
- CSS custom properties for theming
- Flexbox and Grid for layouts
- Mobile-first media queries
- Smooth animations with CSS transitions
- Box-shadow and border-radius for modern look
